SaaSアプリの管理運用を実施するアプリケーションを構成しようと企画している。
今営業部門で管理しているExcel票のみで管理されているが、これをアプリケーション化して管理運用の効率化を図ることが目的である。
このアプリケーションは、SaaSアプリの情報を管理する機能と、SaaSアプリの利用状況を分析する機能を提供する予定である。
SaaSアプリの情報管理機能では、アプリケーションの名称、提供元、利用開始日、利用目的、利用者数などの情報を管理することができるようにする。
SaaSアプリの利用状況分析機能では、アプリケーションの利用状況をグラフや表で表示し、利用者数の推移や利用目的の傾向などを分析できるようにする予定である。
このアプリケーションは、営業部門の担当者がSaaSアプリの管理運用を効率化するためのツールとして活用されることを想定している。
.NETを取り扱う専門部隊でFrontendはReact、BackendはASP.NET Core Web APIで構築することを想定している。

会話の途中で企画書としてPowerPointの資料を作成し、その資料に対してレビューを行うというシナリオ。

企画は新製品開発チームの製品企画チャネルで実施される。
