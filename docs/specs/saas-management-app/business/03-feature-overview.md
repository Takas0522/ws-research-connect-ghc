# 機能概要

## 機能一覧

![機能一覧](../images/image4.png)

本アプリの機能は大きく**顧客情報管理**と**利用状況分析**の2領域に分かれる。

### 顧客情報管理

- 顧客コード・顧客名の登録
- 利用開始日・終了日の管理
- 従量課金（単価・数量）管理
- 担当者の割り当て
- 一覧・検索・フィルタ機能

### 利用状況分析

- 利用者数の推移グラフ
- 利用目的の傾向分析
- ダッシュボード表示
- 月次レポート出力
- CSVエクスポート機能

## 画面構成

![画面構成案](../images/sekkei_image2.png)

4つの主要画面で構成される。

### 1. ダッシュボード

- 顧客別利用状況一覧
- 超過アラート通知（使用率閾値超過時の警告表示）
- 月次トレンドグラフ
- サブ画面：利用率チャート、アラート一覧

### 2. マスタ管理

- 製品マスタ CRUD（SaaS製品の登録・編集・削除）
- 課金プランマスタ CRUD（プラン別料金・上限値の管理）
- メトリクス定義マスタ CRUD（従量課金指標の定義管理）
- サブフロー：一覧 → 詳細 → 編集

### 3. 契約管理

- 顧客別契約情報管理（契約内容・ステータス管理）
- 更新時期アラート（契約更新期限の事前アラート）
- ライセンス数・プラン変更履歴
- 契約ステータス管理

### 4. 従量データ取込

- CSVアップロード機能（月次従量データのCSVアップロード）
- 取込結果プレビュー
- エラーチェック・差分確認（取込データの整合性チェック）
- 将来的にAPI自動連携も想定
