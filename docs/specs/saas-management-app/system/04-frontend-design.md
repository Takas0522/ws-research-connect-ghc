# フロントエンド画面設計

## ページ構成

```
/                       → ダッシュボード（トップページ）
/products               → 製品一覧
/products/:id           → 製品詳細（プラン一覧を含む）
/customers              → 顧客一覧
/customers/:id          → 顧客詳細（契約・利用状況を含む）
/contracts              → 契約一覧
/contracts/:id          → 契約詳細（変更履歴を含む）
/usages                 → 利用実績一覧・登録
/trials                 → トライアル一覧・管理
```

## 共通レイアウト

```
┌─────────────────────────────────────────────────────────┐
│  SaaS Management    [Dashboard] [製品] [顧客] [契約]     │
│                     [利用実績] [トライアル]                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  (ページコンテンツ)                                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

- ナビゲーションは画面上部の水平ナビバー
- Tailwind CSS v4 でスタイリング
- レスポンシブ対応（モバイルはハンバーガーメニュー）

## 画面詳細

### ダッシュボード（/）

トップページ。事業全体の俯瞰ビュー。

```
┌─────────────────────────────────────────────────┐
│  ダッシュボード                                   │
├─────────────┬─────────────┬─────────────────────┤
│  月次売上    │  契約数      │  トライアル          │
│  ¥523,830   │  15 件      │  3 件（進行中）       │
│  (+2.3%)    │  (+1)       │  転換率: 75%         │
├─────────────┴─────────────┴─────────────────────┤
│                                                 │
│  [月次売上推移グラフ]                              │
│  折れ線グラフ（製品別色分け・6ヶ月分）              │
│                                                 │
├─────────────────────┬───────────────────────────┤
│  製品別売上比率       │  期限間近のトライアル       │
│  [円グラフ]          │  ● BizFlow / 新規A  3日   │
│                     │  ● SecureGate / 新規B 5日  │
├─────────────────────┴───────────────────────────┤
│  顧客別 月額ランキング                            │
│  1. XYZホールディングス  ¥152,600                 │
│  2. みどり銀行           ¥93,670                  │
│  3. 田中電機             ¥57,108                  │
└─────────────────────────────────────────────────┘
```

**表示データ**:
- 月次売上合計（前月比）
- アクティブ契約数
- トライアル KPI（進行中件数・転換率）
- 月次売上推移（折れ線グラフ、製品別）
- 製品別売上比率（円グラフ）
- 期限間近のトライアル（残7日以内）
- 顧客別月額ランキング

### トライアル一覧（/trials）

```
┌─────────────────────────────────────────────────┐
│  トライアル管理                  [+ トライアル開始] │
│  フィルタ: [ステータス▼] [顧客▼]                   │
├─────────────────────────────────────────────────┤
│  顧客名      製品        期間          残日数 状態 │
│  ─────────────────────────────────────────────  │
│  新規A      BizFlow    3/1〜3/31      9日  ●進行│
│  新規B      SecureGate 3/10〜4/10    19日  ●進行│
│  ...                                            │
│                                                 │
│  [本契約に転換]  [キャンセル]                      │
└─────────────────────────────────────────────────┘
```

残日数の色分け:
- 7日以上: `text-green-600`
- 3〜6日: `text-yellow-600`
- 1〜2日: `text-red-600`
- 0日以下: `text-gray-400`（期限切れ）

## コンポーネント設計

### 共通コンポーネント

| コンポーネント | 用途 |
|--------------|------|
| `Layout` | 共通レイアウト（ナビバー + コンテンツ領域） |
| `DataTable` | ソート・フィルタ可能なテーブル |
| `FormField` | フォーム入力フィールド（ラベル + バリデーション表示） |
| `Modal` | モーダルダイアログ（登録・編集フォーム用） |
| `StatusBadge` | ステータス表示バッジ（色分け） |
| `Card` | ダッシュボード KPI カード |
| `Chart` | recharts ラッパー（折れ線・棒・円グラフ） |

### 状態管理

- React 19 の `useState` / `useEffect` でのフェッチ
- 各ページで独立したデータ取得（グローバルストアは v1 では不要）
- ローディング状態とエラー状態を各ページで管理
